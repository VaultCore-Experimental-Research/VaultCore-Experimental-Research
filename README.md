# VaultCore Experimental Research

**Experimental RISC-V Systems Research Laboratory**

A self-directed initiative exploring and advancing the RISC-V ecosystem through architecture design, tools, simulators, compilers, kernels, debuggers, and low-level systems experimentation.

## Pinned Projects

- **[rvx-llvm](https://github.com/VaultCore/rvx-llvm)** — RISC-V LLVM Compiler Backend
- **[xtor](https://github.com/VaultCore/xtor)** — x86 to RISC-V Binary Translator

## About This Lab

VaultCore is the personal research workspace of Chrinovic Mukanya (@mu_chrinovic), focused on deep systems programming, computer architecture, and open RISC-V innovation.
